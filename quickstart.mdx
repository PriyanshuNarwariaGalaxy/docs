## Prerequisites

Before you begin, make sure you have:

<Steps>
  <Step title="Galaxy AI Account">
    Sign up at [Galaxy AI](https://workflow-priyanshu-y2um.vercel.app) to get your account.
  </Step>
  <Step title="API Access Token">
    Generate an API token from your dashboard settings.
  </Step>

## Make Your First API Call

### 1. Authentication

All API requests require a Bearer token in the Authorization header:

```bash
Authorization: Bearer YOUR_API_TOKEN
```

### 2. List Your Workflows

Let's start by listing all your workflows:

<CodeGroup>

```bash cURL
curl --request GET \
  --url https://workflow-priyanshu-y2um.vercel.app/api/openapi/workflows \
  --header 'Authorization: Bearer YOUR_API_TOKEN'
```

```javascript JavaScript
const response = await fetch('https://workflow-priyanshu-y2um.vercel.app/api/openapi/workflows', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_TOKEN',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(data.workflows);
```

```python Python
import requests

response = requests.get(
    'https://workflow-priyanshu-y2um.vercel.app/api/openapi/workflows',
    headers={'Authorization': 'Bearer YOUR_API_TOKEN'}
)

print(response.json())
```

</CodeGroup>

### 3. Create a Workflow

Create your first workflow:

```bash
curl --request POST \
  --url https://workflow-priyanshu-y2um.vercel.app/api/openapi/workflows \
  --header 'Authorization: Bearer YOUR_API_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "My First Workflow",
    "nodes": [],
    "edges": []
  }'
```
